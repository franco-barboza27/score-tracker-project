## How to use the Project
***
1. Open the main.py file
2. Press the run button
3. If you have an existing account, select Login. If you don't have an existing account, select Sign Up
4. Sign in to the program
5. Choose an action to do. You can select a game to play or look at the leaderboard, which displays the top 10 scores.

## List of Key features
***
- There are four different games to choose from: arithmetic quiz game (easy and hard mode), number guessing game (easy and hard mode), rock paper scissors, and tic tac toe.
- There are eight different leaderboards for the different games. They show the top 10 scores. (The easy and hard modes of both the arithmetic quiz game and the number guessing game count as different categories.)
- Data is saved between users, and passwords are hashlibbed for security.


## Installation instructions
***
As long as the Programming drive is accessible, this program should work.
